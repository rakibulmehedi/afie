## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: rakibulmehedi.dev@gmail.com
Subject line: `[SECURITY] afie — <brief description>`

**Response SLA:**
- Acknowledgement: within 72 hours
- Status update: within 7 days
- Fix or mitigation plan: within 30 days for critical issues

Include: affected component, reproduction steps, potential impact, and any suggested fix.
